# Maaja Bot 🚀

An AI-powered stock trading assistant built using Python.

## 🔍 Features
- Scans multiple stocks
- Uses RSI, EMA, Volume
- Generates signals
- Sends Telegram alerts

## 🛠 Tech Stack
- Python
- APIs
- Automation

## 📈 Goal
To build an automated trading system.

## Status
In development

## 🚀 Advanced Features
- Live stock data using Yahoo Finance API
- RSI-based signal generation
- Telegram alert integration
